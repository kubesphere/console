# Icon 组件

图标组件，用于显示矢量图标。

## 示例

```jsx live=true
() => {
  return (
    <div>
      <Icon name="cluster" color="green" size={24} />
      <Icon name="role" color="red" size={24} />
    </div>
  );
}
```

在这个示例中，我们使用 `Icon` 组件来显示不同的图标。我们传递了 `name` 属性来指定图标的名称，`color` 属性来设置图标的颜色，`size` 属性来设置图标的大小。组件将会渲染相应的图标，并显示在页面上。
