# LabelText 组件

用于设置标签文本样式的组件。

## 示例

```jsx live=true
() => {
  return (
    <LabelText>
      这是一个标签文本样式的示例。
    </LabelText>
  );
}
```

在这个示例中，我们使用 `styled-components` 库创建了一个名为 `LabelText` 的组件，它基于 `Text` 组件，并设置了特定的样式属性。你可以将这个组件用作标签文本的样式化容器。
