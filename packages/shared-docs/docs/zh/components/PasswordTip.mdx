# PasswordTip 组件

密码提示组件，用于显示密码的强度和要求。

## 示例

```jsx live=true
() => {
  const password = "Password123";
  const className = "custom-password-tip";
  const hasProgress = true;

  return <PasswordTip password={password} className={className} hasProgress={hasProgress} />;
}
```

在这个示例中，我们使用 `PasswordTip` 组件创建一个密码提示组件。我们传递了一个 `password` 属性，该属性表示输入的密码。我们还传递了一个 `className` 属性来自定义样式类名，并通过 `hasProgress` 属性控制是否显示密码强度进度条。组件将会根据输入的密码计算密码强度并显示相应的要求和进度条。
