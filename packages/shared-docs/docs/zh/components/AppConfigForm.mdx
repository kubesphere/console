# AppConfigForm 组件

用于显示和编辑应用配置的表单组件。

## 示例

```jsx live=true
() => {
  const formRef = useRef(null);

  const handleFormSubmit = () => {
    if(formRef.current) {
      console.log(formRef.current.conf)
    }
  };

  return (
    <div>
      <AppConfigForm ref={formRef} />
      <button onClick={handleFormSubmit}>提交</button>
    </div>
  );
}
```

在这个示例中，我们有一个 `AppConfigForm` 组件，用于显示和编辑应用的配置信息。

你可以直接使用 `AppConfigForm` 组件，并通过 `ref` 来访问表单数据并验证字段。
